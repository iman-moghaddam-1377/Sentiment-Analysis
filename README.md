## 📖 Project Overview
The objective is to design a regression model that predicts a numerical score for product reviews. The system processes raw text data and outputs a single continuous value representing the predicted rating.

### 🛠️ Key Components
*   **Preprocessing**: Comprehensive data cleaning and preparation of the review dataset.
*   **Pre-trained Embeddings**: Utilization of **GloVe** (Global Vectors for Word Representation) as the initial embedding layer.
*   **Model Architecture**: 
    *   Initial implementation using **LSTM** blocks.
    *   Comparative implementation using **BiLSTM** (Bidirectional LSTM) blocks.
*   **Task Type**: Regression (the network outputs a single numerical value).
